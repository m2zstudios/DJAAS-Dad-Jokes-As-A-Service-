# Dad-Jokes-as-a-Service (DJaaS)

Run:
1. npm install
2. node index.js
3. Open http://localhost:3000/joke

Endpoints:
- GET /joke         -> single joke
- GET /jokes?count=5 -> multiple jokes
- GET /joke?type=hinglish -> type filtered
- GET /meta         -> meta info

- Ads on landing page
